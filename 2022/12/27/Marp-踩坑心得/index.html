還在為著 PPT 要調圖片調一整天而困擾嗎？還在為著不知道要怎麼排版比較好嗎？推薦 Marp，它超好用的！

keywords: Marp

安裝

  • 在 vscode 上面下載 Marp for VS Code 套件

自定義 Theme

  • 下載別人寫好的 css 檔案

  • 將目標投影片的 .md 檔與 css 檔案放在同一個資料夾底下

  • 並使用 vscode 開起對應資料夾

  • 按下 ctrl + , 進入設定，在搜尋列輸入：marp: theme

  • 按下新增項目，輸入 css 的檔案路徑 (記得要加 ./)

  • 享受別人的成果吧！

  •   ---
  marp: true
  theme: olive
  ---
